# parking
Aplikacja desktopowa do zarządzania parkingiem.

Funkcjonalności: 
* Rezerwacja miejsca
* Usunięcie pojazdu
* Wjazd pojazdu
* Wyjazd pojazdu
* Sprawdzenie ilości naliczonych opłat
* Uiszczenie opłaty
* Wyświetlenie listy pojazdów 
* Zapisanie stanu do obiektu
